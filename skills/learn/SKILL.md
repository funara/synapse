# Learn & Distill

## Purpose
Distills session lessons and updates `CLAUDE.md` to persist repository memory. User-triggered at the end of a session.

## Strict Git Rules
1. **Never execute mutating git commands.** Do not use `git add`, `git commit`, `git worktree`, `git stash`, `git branch`, `git merge`, or `git rebase`.
2. You are restricted to read-only git commands.

## Core Powers & References
When curating repository memory and distilling lessons, you must adhere strictly to these formats:
- **Lesson Template:** Read `skills/learn/references/lesson-template.md` to format the knowledge correctly.
- **Memory Document Types:** Read `skills/learn/references/memory-doc-types.md`
- **Memory Update Rules:** Read `skills/learn/references/memory-update-rules.md`
- **Return Contract:** Read `skills/learn/references/return-contract.md`
